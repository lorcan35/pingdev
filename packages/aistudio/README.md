# PingApp: aistudio

> Auto-generated PingApp shim for [https://aistudio.google.com](https://aistudio.google.com)

**Category:** chat
**Purpose:** Google AI Studio - AI chat playground and prompt engineering with Gemini models

## Actions

- **sendMessage** (primary): Type a prompt and submit it to Gemini. Uses Ctrl+Enter or the Run button to submit.
- **newConversation**: Start a new chat conversation. Navigate to /prompts/new_chat or click the New chat button.
- **switchModel**: Change the active model by clicking the model selector card in the run settings panel.
- **setSystemInstructions**: Open system instructions panel to set system prompt for the model.
- **toggleTool**: Toggle a tool on/off (structured outputs, code execution, function calling, grounding, URL context) via their switch controls.
- **setThinkingLevel**: Set the thinking level (Low/Medium/High) via the thinking level combobox.
- **setTemperature**: Adjust the temperature slider/input in the run settings panel.
- **addMedia**: Open the media insertion menu to attach images, videos, audio, or files.
- **getCode**: Open the Get Code panel showing API code for the current prompt/settings.
- **stopGeneration**: Stop the current response generation.

## States

- **idle**: Chat input is visible and enabled, no stop button visible, submit button may be disabled (empty input) → typing
- **typing**: Chat input has content, submit button becomes enabled → generating, idle
- **generating**: Stop button is visible, response area is being populated → done, error, idle
- **done**: Stop button disappeared, response text is stable (hash_stability), response container has content → idle
- **error**: Error message visible in response area, or network error toast notification → idle
